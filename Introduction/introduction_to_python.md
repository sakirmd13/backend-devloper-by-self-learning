# 📖 What is Python?

Python is a **high-level, interpreted, object-oriented, and general-purpose programming language** created by **Guido van Rossum** and first released in **1991**.

Python focuses on **readability** and **simplicity**, allowing developers to write fewer lines of code compared to many other programming languages.

Example:

```python
print("Hello, World!")
```
